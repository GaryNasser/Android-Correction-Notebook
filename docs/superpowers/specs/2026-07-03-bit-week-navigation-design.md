# BIT 课表周导航设计

日期：2026-07-03

## 目标

让 `BIT` 课表页像真正课表一样支持周切换和日期选择。

## 范围

- 一页仍显示 7 天
- 每页固定为自然周：周一到周日
- 支持上一周、下一周
- 支持点击日期范围打开日期选择器
- 选择任意日期后跳到该日期所在自然周
- 支持一键回到本周

## 状态规则

- `HomeUiState.selectedDate` 在 `BIT` 周课表中表示当前周的周一
- `ScheduleRange.WEEK` 继续表示从 `selectedDate` 起 7 天
- 进入 `BIT` 页面时保持当前课表周，不强制重置
- 首次打开时默认当前自然周

## 验收标准

- 课表顶部显示类似 `7月1日 - 7月7日`
- 左箭头切到上一自然周
- 右箭头切到下一自然周
- 日期范围可点击并打开日期选择器
- 选择日期后显示该日期所在的周一到周日
- 今天按钮回到当前自然周
